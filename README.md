# 🌍 Language Translation Tool (with Voice Support)

## 📖 Project Description
This is a simple and advanced Language Translation Tool developed using Python.

The tool not only translates text from one language to another but also supports **voice input and voice output**, making it more interactive and user-friendly.

Users can speak input text, get instant translation, and also hear the translated output using text-to-speech.

This project was developed as part of Code Alpha / Lodhi internship or learning practice.

---

## ✨ Features
- 🌐 Translate text between multiple languages  
- 🎤 Voice input (speech-to-text) support  
- 🔊 Voice output (text-to-speech) support  
- ⚡ Fast and accurate translation  
- 🧠 Easy to use for beginners  
- 💻 Simple and interactive interface  

---

## 🛠️ Technologies Used
- Python  
- Googletrans / deep-translator library  
- SpeechRecognition (for voice input)  
- pyttsx3 (for voice output)  
- Tkinter (if GUI used)
